## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run Lint
```bash
$ npm run lint
```

## Run tests
```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Run a Docker for MongoDb
```bash
#Start the MongoDB container
$ docker-compose up -d

#Check if the MongoDB container is running
$ docker-compose ps

#Stop the MongoDB container
$ docker-compose down
```
